# 01_advanced_basics

## Log Analyzer

### Описание
Анализатор логов nginx.

### Параметры конфигурации
* REPORT_SIZE - количество url-ов, которые будут выведены в отчет. По умолчанию 1000.
* REPORT_DIR - Директория для сохранения отчетов. По умолчанию "./reports".
* LOG_DIR - Директория для поиска логов. По умолчанию "./log".
* LOG_FILE - Файл логов работы анализатора. По умолчанию не задан, вывод логов осуществляется в стандартный поток (stdout).
* CONFIG_PATH - Путь к файлу конфигурации. По умолчанию "config.ini".
* LOGGING_LEVEL - Уровень логгирования работы анализатора. По умолчанию "INFO".
* ALLOW_ERROR_RATE - Допустимый процент ошибок при парсинге (отношение ошибочных строк к корректным). По умолчанию "0.1".

### Запуск
Запуск с конфигурацией по умолчанию:
>python log_analyzer.py

Запуск с конфигурацией, из файла, указанного в CONFIG_PATH:
>python log_analyzer.py --config

Запуск с конфигурацией, из файла config.ini:
>python log_analyzer.py --config config.ini


### Выполнение тестов

>python test_log_analyzer.py
